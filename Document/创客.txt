请求url  http://metmastersandbox.metworld.com.cn:8093/met/team  post json传输

1、队伍列表
请求参数
{
	type 1 
	page 当前页数 1开始
	num 每页显示数量
}
返回值{
	total 总数
	array {
		Json{
			id 创客id
			name 创客名字
			icon 创客icon
			desc 创客描述
			sign 创客口号
			score 权重，越大越靠前
			isMaker 是否是创客
			time 创建时间
		}
	
	}
}

2、单个队伍
请求参数
{
	type 2 
	id 创客id
}
返回值{
	
	id 创客id
	name 创客名字
	icon 创客icon
	desc 创客描述
	sign 创客口号
	score 权重，越大越靠前
	bool 是否是创客
	time 创建时间

	members 数组{
		Json{
			id 队员id
			position   1队长2副队长3队员
			name 队员名字
			icon 队员icon
			profession 队员职业
			desc 描述
			age 年龄
			sex 
		}
	}
}

3、删除创客
请求参数
{
	type 3
	id 创客id
}
返回值{
	id 创客id
}
