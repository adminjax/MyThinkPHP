请求url  http://metmastersandbox.metworld.com.cn/met/video:80853  post json传输


1、已经公开的视频List
请求参数{
	page 当前页数1开始
	num 返回数量	 
	type 1
}
返回值{
	size 公开视频总数量
	array jsonArray{
			jsonObject{
				id 视频id
				time 创建时间
				title 标题
				pic 视频封面
				url 视频连接地址
				pname 发布者名字（-----名字会有重复）
				pid 发布者pid 唯一
			}
	}
}
